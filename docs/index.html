ngoy

ngoy is a template engine for the JVM, based on the Angular+ component architecture.

ngoy is a pure Java library with no JavaScript involved. Everything is server side.

ngoy is designed as a generic template engine that allows to produce any text format. HTML is just one of them.

enjoy ngoy!
